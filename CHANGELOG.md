# Changelog

All notable changes to the Proposal Development Framework will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-04

### Added
- Initial release of Proposal Development Framework
- 6-step proposal workflow (discover, scope, plan, budget, compliance, compile)
- AGENTS.md with comprehensive command definitions
- README.md with framework overview and usage guide
- Detailed task instruction files:
  - 01-discover-opportunity.md - Discovery interview and requirements gathering
  - 02-create-scope-of-work.md - Scope definition and deliverables
  - 03-generate-project-plan.md - Project planning and scheduling
  - 04-create-budget.md - Comprehensive budgeting and cost breakdown
  - 05-compliance-risk.md - Compliance documentation and risk management
  - 06-compile-proposal.md - Final proposal assembly and compilation
- Document management commands (executive summary, PDF export, budget summary)
- Quality commands (review, compliance check, price comparison)
- Archive and tracking commands
- Checkpoint and restart capabilities
- Folder structure for proposals, checkpoints, templates, and company data

### Changed
- Adapted from software development methodology to construction/infrastructure proposals
- Replaced technical artifacts (SRS, PRD, tech stack) with proposal-specific documents
- Swapped dev commands (build, test, deploy) for proposal commands

### Notes
- Framework designed for construction, infrastructure, and consulting proposals
- Supports interactive interview process for proposal development
- Includes templates for company profiles, pricing databases, and compliance requirements
- Original software development framework archived in /archive folder
