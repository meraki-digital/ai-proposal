# Templates

This folder contains reusable templates, pricing databases, and standards.

## Recommended Files

### proposal-template.md
Standard proposal formatting, cover page layout, section structure

### pricing-database.xlsx
Historical cost data:
- Labor rates by role
- Equipment hourly/daily rates
- Material unit costs
- Subcontractor pricing
- Overhead rates

### compliance-requirements.md
Standard compliance requirements by:
- Industry (construction, infrastructure, consulting)
- Region (state/local regulations)
- Client type (public sector, private, federal)

### terms-and-conditions.md
Standard contract terms, payment terms, warranty language

### branding-guide.md
Logo usage, color schemes, fonts for proposal documents

### insurance-requirements.md
Standard insurance coverage levels and requirements

## Usage

Templates are referenced throughout the proposal workflow:
- Step 4 (Budget) uses pricing-database.xlsx
- Step 5 (Compliance) uses compliance-requirements.md
- Step 6 (Compile) uses proposal-template.md

## Privacy

This folder is git-ignored by default to protect proprietary pricing and templates.

## Best Practices

- Keep pricing data current
- Update after each major project with actual costs
- Maintain separate pricing for different project types or regions
- Version templates when making significant changes
