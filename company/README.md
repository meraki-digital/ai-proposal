# Company Data

This folder contains company-specific information used across all proposals.

## Recommended Files

### profile.md
Company overview, history, size, locations, contact information

### capabilities.md
Core competencies, services offered, technical capabilities, equipment inventory

### past-projects/
Folder with case studies and examples of completed projects:
```
past-projects/
├── project-001-description.md
├── project-002-description.md
└── photos/
```

### certifications.md
Certifications, licenses, memberships, awards, safety ratings

### personnel/
Key personnel bios and qualifications (for proposal team sections)

### financial.md
Bonding capacity, financial strength indicators (if shareable)

## Usage

These files are referenced by proposal steps:
- Step 2 (Scope) references capabilities.md
- Step 6 (Compile) pulls from profile, past projects, certifications

## Privacy

This folder is git-ignored by default. Add files here without committing proprietary company information to the repository.
